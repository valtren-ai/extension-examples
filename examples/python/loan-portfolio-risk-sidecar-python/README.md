# loan-portfolio-risk-sidecar-python

Micro-finance Python sidecar for borrower and portfolio risk scoring.

## What it demonstrates

- `/health` for runtime checks
- `/portfolio/risk` for delinquency and utilization-based borrower scoring
- a practical Python starting point for credit risk and collections teams

## Example use case

A digital lender wants to prioritize borrowers for intervention using days past due, utilization, and recent missed-payment behavior.
