# org-collections-zip-python

Minimal Python org ZIP extension for collections and recovery teams.

## What it demonstrates

- `app.py` at ZIP root
- `/health` route for Valtren smoke test
- `/collections/review` endpoint for simple collections-lane routing

## Best for

- customer-owned collections proof of concept
- org ZIP upload through `Admin -> Org Settings -> Extensions`
- quick servicing and collections experiments without sidecar infrastructure

## How to zip it

Zip the contents of this folder, not the folder itself.
