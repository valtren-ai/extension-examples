# org-telecom-retention-zip-python

Minimal Python org ZIP extension for telecom retention and renewal teams.

## What it demonstrates

- `app.py` at ZIP root
- `/health` route for Valtren smoke test
- `/retention/review` endpoint for simple churn-retention routing

## Best for

- telecom-owned proof of concept
- retention and renewal workflows
- org ZIP upload through `Admin -> Org Settings -> Extensions`

## How to zip it

Zip the contents of this folder, not the folder itself.
