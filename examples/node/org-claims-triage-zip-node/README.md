# org-claims-triage-zip-node

Minimal Node org ZIP extension for insurance claims triage teams.

## What it demonstrates

- `index.js` at ZIP root
- `/health` route for Valtren smoke test
- `/claims/triage` endpoint for simple claims-review recommendations

## Best for

- insurer-owned proof of concept
- claims triage and manual-review routing
- org ZIP upload through `Admin -> Org Settings -> Extensions`

## How to zip it

Zip the contents of this folder, not the folder itself.
