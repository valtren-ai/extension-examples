# steel-plant-ops-pack

Manufacturing-focused Node extension for steel plant workflows.

## What it demonstrates

- a domain-specific workflow pack for plant operations
- a custom Node executor that prepares a next intervention step
- a starter template that feels relevant to steel/plant users rather than generic AI samples

## Good fit for

- steel plants
- rolling mills
- melt shop / furnace supervision
- plants that want governed intervention loops with approvals

## Example scenario

Use this pack when you want an agent to detect rolling-mill quality drift or furnace instability, route approval to a shift lead, and prepare the next recommended intervention.
