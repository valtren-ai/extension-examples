# work-order-risk-sidecar-dotnet

Manufacturing-focused .NET sidecar for maintenance work-order risk scoring.

## What it demonstrates

- `/health` for runtime checks
- `/work-order/risk` for breakdown and delay-based risk scoring
- a practical .NET example for maintenance, reliability, or planning teams

## Example use case

A plant wants to flag delayed breakdown work orders that may threaten production continuity or safety.
