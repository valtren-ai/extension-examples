# furnace-energy-sidecar-java

Manufacturing-focused Java sidecar for furnace energy drift analysis.

## What it demonstrates

- `/health` for runtime checks
- `/furnace/energy` for furnace energy deviation analysis
- a simple Spring Boot base for energy, utilities, or process teams

## Example use case

A steel plant wants to detect when a furnace or heat-treatment line is consuming more energy than expected per ton and route a corrective review.
