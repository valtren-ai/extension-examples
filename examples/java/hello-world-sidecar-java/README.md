# hello-world-sidecar-java

Simple Java sidecar example for Valtren AI using Spring Boot.

## What it demonstrates

- `/health`
- `/hooks/{hookName}`
- a minimal sidecar contract in a JVM-friendly stack

## Local run

```bash
cd examples/java/hello-world-sidecar-java
./mvnw spring-boot:run
```

Or run it with your installed Maven:

```bash
mvn spring-boot:run
```
